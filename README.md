## üìñ Overview
This project is a comprehensive, interactive reference tool designed for Forensic Engineers and Incident Responders. It transforms complex forensic data into a streamlined, navigable experience, mapping specific Windows artifacts to investigative workflows.

The guide covers everything from **Program Execution** to **External Device Tracking**, providing deep technical specifications, registry paths, and evidentiary value for each artifact.

## üõ†Ô∏è Key Features
- **Interactive UI:** A Single Page Application (SPA) with a sidebar navigation for seamless transitions.
- **Deep Technical Specs:** "Click-to-reveal" mechanics for deep-dive technical data (Paths, GUIDs, ROT-13, etc.).
- **Forensic Mind-Map Logic:** Organized categories following the investigative lifecycle.
- **Modern Aesthetics:** A professional Dark/Blue palette optimized for long research sessions.

## üóÇÔ∏è Forensic Modules Covered
| Module | Description | Key Artifacts |
| :--- | :--- | :--- |
| **01. Execution** | Proving code was triggered. | UserAssist, Shimcache, Amcache, Prefetch, BAM. |
| **02. File Knowledge** | Evidence of user intent. | ShellBags, LNK Files, Jump Lists, Open/Save MRU. |

| **06. Cleanup** | Anti-forensics detection. | Recycle Bin ($I / $R), Gap Analysis. |
